# ☕ Coffee Sales Dashboard (Excel)

An interactive **Coffee Sales Dashboard** built in Microsoft Excel to analyze coffee shop sales, customer footfall, and product performance using Pivot Tables, Pivot Charts, and Slicers.

## 📊 Dashboard Preview

![Dashboard Screenshot](Screenshot%202026-07-06%20174842.png)


---

## 🚀 Project Overview

This dashboard helps analyze coffee shop performance by providing insights into:

- 💰 Total Sales
- 👥 Total Footfall
- 📈 Average Bill per Person
- ☕ Average Orders per Person
- ⏰ Quantity Ordered by Hour
- 🥧 Category-wise Sales Distribution
- 📦 Size-wise Order Distribution
- 🏪 Sales by Store Location
- ⭐ Top 5 Best-Selling Products
- 📅 Orders by Day of the Week
